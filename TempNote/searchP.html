问题描述

魔方是一种常见的益智游戏，如下图是魔方的一个图片:

图1. 二阶、三阶、四阶、八阶魔方

魔方根据其面上正方体的个数，可以分为多种魔方，上面的魔方从左到右分别是：2阶魔方、3阶魔方、4阶魔方和8阶魔方，通常 2 ~ 11 阶的魔方称为低阶魔方，高于 11 阶的魔方称为高阶魔方 。

魔方是一个正方体，共有6个面，每个面上的小正方形个数与魔方的阶数相关。例如，3 阶魔方的一个面上有 3×3=9 个正方体，每个正方体的颜色有 6 种可能。

我们需要解决的问题描述起来非常简单：将一种可能的魔方状态还原为每一个面上的颜色都相同的状态。这里所解决的问题主要是针对 3 阶魔方的求解过程，后面在扩展部分会对 3 阶以上的魔方解决思路进行进一步探讨。

状态空间分析

该问题是已知初始状态和终止状态，求解一条路径的搜索问题。其中初始状态是打乱的状态，如上图1所示，终止状态为一个有序的正方体，如下图所示：

根据维基百科上的说明，三阶魔方的总变化数是：
8!×38×12!×2122×2×3=43252003274489856000≈4.33×1019\frac{8!\times3^8\times12!\times2^{12}}{2\times2\times3}=43252003274489856000\approx4.33\times10^{19}2×2×38!×38×12!×212​=43252003274489856000≈4.33×1019

三阶魔方总数的算式是这样得到的：

  • 8个角块可以互换位置（8!），也可以旋转（3），但不能单独翻转一个角块，所以总共有8!×38/3种变化状态。
  • 12个边块可以互换位置（12!），也可以翻转（2），但不能单独翻转一个棱块（也就是将其两个面对调），也不能单独交换两棱块的位置，所以总共有12!×212/(2×2)12! \times 2^{12} / (2×2)12!×212/(2×2)种变化状态

而与一个状态相邻的状态表示只旋转一次所得到的状态，共有 12 种，分别为旋转 上、下、左、右、前、后 6个面，乘上 正转、反转两种旋转方式。

因此，三阶魔方的状态空间为 4.33×10194.33\times10^{19}4.33×1019，每个状态有 6 个后继状态，初态为任意的一个状态，目标测试 Goal 为所有面上的正方形颜色相同，路径耗散为从初态到通过目标测试状态所进行状态改变次数。

class Block(object):
    ID = 0
    def __init__(self, cols):
        self.id = Block.ID
        Block.ID += 1
        self.cols = cols
    def rotate(self, direction, bits=1):
        cols = []
        for i in BLOCKCHANGE[direction]:
            cols.append(self.cols[i])
        for i in range(4):
            index = (i + bits) % 4
            self.cols[BLOCKCHANGE[direction][index]] = cols[i]
        # for i in BLOCKCHANGE[direction]:
        #     cols.append(self.cols[i])
        # print self.id, cols
    def getCol(self, side):
        return self.cols[side]
    def setCol(self, side, col):
        self.cols[side] = col

from bidict import bidict
import copy
import re

class Magic(object):
    # 上(红) 下(橘) 左(蓝) 右(绿) 前(白) 后(黄)
    color_dir = bidict({0:'r', 1:'o', 2:'b', 3:'g', 4:'w', 5:'y'})
    default_blocks = [[COL1, COL2, COL3, COL4, COL5, COL6] for i in range(27)]

    def __init__(self, init_info=None):
        self.blocks = [Block(i) for i in self.default_blocks]
        if init_info != None:
            valid_info = ''.join(re.findall(r'(\S)', init_info))
            top_s = [valid_info[i:i+3] for i in (0, 3, 6)]
            left_s = [valid_info[i:i+3] for i in (9, 21, 33)]
            front_s = [valid_info[i:i+3] for i in (12, 24, 36)]
            right_s = [valid_info[i:i+3] for i in (15, 27, 39)]
            back_s = [valid_info[i:i+3] for i in (18, 30, 42)]
            bottom_s = [valid_info[i:i+3] for i in (45, 48, 51)]
            self.set_face_state(TOP, top_s)
            self.set_face_state(LEFT, left_s)
            self.set_face_state(FRONT, front_s)
            self.set_face_state(RIGHT, right_s)
            self.set_face_state(BACK, back_s)
            self.set_face_state(BOTTOM, bottom_s)

    def get_face_state(self, side):
        tp_str = ''.join(self.color_dir[self.blocks[i].getCol(side)]
            for i in FACES[side])
        return [tp_str[i:i+3] for i in (0,3,6)]
    
    def set_face_state(self, side, states):
        for i,j in zip(FACES[side], ''.join(states)):
            self.blocks[i].setCol(side, self.color_dir.inv[j])

    def __str__(self):
        ret_str = ""
        top_state = self.get_face_state(TOP)
        ret_str += ''.join([' '*3+i+'\n' for i in top_state])
        left_state = self.get_face_state(LEFT)
        front_state = self.get_face_state(FRONT)
        right_state = self.get_face_state(RIGHT)
        back_state = self.get_face_state(BACK)
        ret_str += ''.join([ i1+i2+i3+i4+'\n' for i1,i2,i3,i4 in 
            zip(left_state,front_state,right_state,back_state)])
        bottom_state = self.get_face_state(BOTTOM)
        ret_str += ''.join([' '*3+i+'\n' for i in bottom_state])
        return ret_str

    def rotate(self, side, direction):
        bits = direction
        for i in FACES[side]:
            self.blocks[i].rotate(side, bits)
        blocks = []
        for i in MAGICCHANGE[side]:
            blocks.append(self.blocks[i])
        for i in range(12):
            index = (i + bits * 3) % 12
            self.blocks[MAGICCHANGE[side][index]] = blocks[i]
